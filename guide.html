スマホでの使い方

見る・キーワードを変える・すぐ反映する。やることはこの3つだけです。

結果を見る

いつでも・ログイン不要

  1. Safari で data-board-3710.github.io/board/ を開く
  2. 上の「モニターを開く」ボタンからでもOK
  3. 赤い帯が出たら、誹謗中傷が見つかったサイン。Gmail にも通知が届く
✏️

キーワードを変える

監視する言葉を入れ替える

  1. Safari で github.com/data-board-3710/board/blob/main/keyword.txt を開く（上の「キーワード」ボタンからでもOK）
  2. 聞かれたら data-board-3710 でログイン
  3. 「Code / Blame」の右の ･･･ をタップ
  4. Edit file → In place をタップ
  5. 今の言葉を消して、新しいキーワードを 1 行で入力
  6. 緑の Commit changes をタップ → 確認でもう一度タップ
💡
変えてもすぐには画面に出ません。次の自動巡回（2時間おき）で反映されます。すぐ見たいときだけ、下の「すぐ反映する」を実行してください。
⚡

すぐ反映する（任意）

2時間を待たずに今すぐ巡回

  1. Safari で github.com/data-board-3710/board/actions を開く
  2. x-monitor をタップ
  3. Run workflow をタップ → もう一度タップ
  4. 1〜2分待つ。緑のチェックが出たら成功
  5. モニターを開いて再読み込みすると、新しい結果が出る
💡
急がないなら、ここは使わなくてOK。キーワードを変えておけば、放っておいても2時間おきに自動で巡回します。
困ったとき
  1. 通知が来ない → Gmail の迷惑メールフォルダも確認
  2. 結果が変わらない → 巡回がまだ。少し待つか「すぐ反映する」を実行
  3. 新しい言葉で何も出ない → その言葉は X での投稿が少ないかも。別の言い回しや略称を試す

X監視 → AI判定（誹謗中傷・ネガ・通常・ポジ）→ Gmail通知 → このボードに表示
